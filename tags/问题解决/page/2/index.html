vscode打开django项目pylint提示has not "object" member

vscode 打开 django 项目提示 has not “object” member 是因为 Django 动态地将属性添加到所有模型类中，所以 ide 无法解析。 解决方案： 安装 pylint-django 1 pip install -U pylint-django 启用 pylint-django 打开项目下自动生成的 .vscode 文件夹下的 setting.json 文件，添加下面的配置项。 1 2 3 "python.linting.pylintArgs": [ "--load-plugins=pylint_django" ]
